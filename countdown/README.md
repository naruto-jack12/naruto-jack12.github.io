# ⏳ 精确倒计时

在线精确倒计时工具，精确到秒，支持自定义目标时间和快捷预设。

## 功能

- **精确到秒** — 每秒实时更新，带脉冲动画
- **自定义目标** — 手动选择日期时间
- **快捷预设** — 元旦、春节（1900-2100 全覆盖）、圣诞、1小时后、24小时后
- **进度条** — 渐变进度条显示已过时间占比
- **自动保存** — 数据保存在浏览器本地存储
- **暗黑主题** — 星空背景 + 霓虹风格

## 使用

直接用浏览器打开 `index.html` 即可，无需服务端。
