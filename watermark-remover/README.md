# 图片去水印工具

基于 Web 的纯前端图片去水印工具，无需后端服务，所有处理在浏览器本地完成。

> ⚠️ **注意**：本工具主要为单一颜色背景图片设计，复杂背景图片的处理效果可能不理想。建议选择水印区域后尝试多种去除方式。

## 功能

| 功能 | 说明 |
|------|------|
| **四种选择方式** | 矩形框选、画笔涂抹、魔棒自动选取、仿制图章 |
| **五种去除方式** | 智能填充、纹理填充 (PatchMatch)、快速修复、高斯模糊、AI 修复 (LaMa) |
| **AI 修复** | 基于 LaMa 深度学习的图像修复模型（~207MB，首次使用时按需下载，之后缓存） |
| **撤销 / 重做** | 支持多步撤销与重做 |
| **缩放 / 平移** | 滚轮缩放（以光标为中心）、空格 + 拖拽平移、鼠标中键平移、触屏双指操作 |
| **原图对比** | 按住「对比」按钮查看修复前后效果 |
| **主题切换** | 亮色 / 暗色主题一键切换，偏好自动保存 |

## 快捷键

| 按键 | 功能 |
|------|------|
| `R` | 矩形选择 |
| `B` | 画笔选择 |
| `W` | 魔棒选择 |
| `C` | 仿制图章 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Enter` | 应用当前处理 |
| `Escape` | 清除选区 |
| `Space + 拖拽` | 平移图片 |
| `滚轮` | 缩放图片 |
| `鼠标中键拖拽` | 平移图片 |

## 使用步骤

1. 浏览器打开 `index.html`
2. 点击上传区域或拖入图片（支持 JPG / PNG / WebP）
3. 使用选择工具标记水印区域
4. 选择去除方式
5. 点击应用按钮处理
6. 下载处理后的图片

## 技术栈

- **原生 JavaScript** — 无框架依赖
- **Canvas 2D API** — 图像像素级操作
- **ONNX Runtime Web** — 浏览器端运行 AI 模型（WebGPU / WASM 双后端）
- **LaMa** — 深度学习图像修复模型
- **Cache API** — 离线缓存 AI 模型文件
