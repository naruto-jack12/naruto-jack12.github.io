# 🪷 电子木鱼

敲电子木鱼，积赛博功德。

## 功能

- **点击木鱼** 或按 **空格/回车** 敲击
- 功德计数、连击统计、敲击频率
- 自动积德模式
- 主题切换（暗色 / 浅色 / 禅意绿）
- 木槌动画 + 涟漪光效 + 浮动文字
- 音效（Web Audio API 合成木鱼声）
- 本地存储主题偏好

## 使用

用浏览器打开 `index.html` 即可。
